Telegraph Project

The Telegraph Project showcases how classical data transmission modes were designed using discrete analog currents to express digital codes. By switching copper line circuits closed and open, operators could carry data instantly across continents.

Our implementation breaks down historical Morse-key wiring mechanisms, signal attenuation, and long-range relay structures used to ensure signal clarity over lengthy distances.

Course Overview

📡

1. Intro to Data Comm

  • Meaning: The structural exchange of digital data between a sender and receiver using an arbitrary physical routing medium.
  • Components: Message, Sender, Receiver, Transmission Medium, and Protocols.
  • Importance: Builds the fundamental backbone for the web, telemetry, enterprise communication, and structural IoT networks.
🔌

2. Transmission Modes

  • Simplex: Uni-directional path lines restricting transmission to one direction (e.g., Traditional broadcast radio, keyboard signals).
  • Half-Duplex: Dual-direction paths capable of transferring both ways, but not concurrently (e.g., Walkie-talkies).
  • Full-Duplex: Simultaneous bidirectional link pipelines optimizing data rates (e.g., Cellular networks, VoIP streams).
🌐

3. Network Types

  • LAN: Local Area Network routing confined to close geographical scales (e.g., Offices, Homes).
  • MAN: Metropolitan Area Network bridging medium distances to link multiple city sectors.
  • WAN: Wide Area Networks stretching globally to interlink multiple LAN/MAN groups (e.g., The Internet).
📶

4. Transmission Media

  • Guided Media: Grounded physical wiring infrastructures including Fiber-optic fibers, coaxial lines, and copper twisted pairs.
  • Unguided Media: Wireless signal projections routing data using radio waves, cellular microwave links, or short-range infrared beams.
🧠

5. Network Devices

  • Routers & Switches: Smart packet-routing equipment that routes frames safely through local nodes or global domains.
📦

6. Data Encoding & Signals

  • Analog vs Digital: Continuously fluid waves vs discrete binary step-voltages representing numeric bits.
  • Modulation: Transforming simple data streams into specialized signal properties using Amplitude, Frequency, or Phase techniques.
🔐

7. Network Protocols
